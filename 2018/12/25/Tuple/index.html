使用Tuple打包数据

发表于 2018-12-25 | In Swift 基础知识 |

很多时候，我们需要把多个不同类型的值，打包成一个单位处理。例如，返回一个HTTP状态信息：

  • 状态码: 200; 状态消息: HTTP OK
  • 状态码: 404; 状态消息: File not found
    或者，返回一条数据库用户信息记录：姓名: Mars; 工号: 11; 电子邮件: 11@boxue.io。在Swift里，我们可以使用Tuple 来很方便的处理类似的问题。
